# ProjInvest Web

Sistema web para gestão de carteira de investimentos integrado com PostgreSQL.

## 🚀 Status

✅ **MVP funcional**  
- Dashboard com KPIs  
- CRUD Clientes, Contas, Ativos, Posições  
- Importação de extrato B3 (CSV)  
- Integração PostgreSQL completa  

## 📋 Funcionalidades

- Listagem e navegação por todas as entidades do domínio
- Dashboard com contadores
- API de upload B3 com deduplicação
- Layout responsivo com menu lateral
